# ADR-001: Visión y Alcance de software-tools v2

## Estado
Aprobado

## Contexto
El proyecto software-tools operó entre 2005-2018 como catálogo de descargas para cursos Oracle. En 2025 se decide evolucionarlo hacia un archivo histórico personal de herramientas utilizadas en 20+ años de docencia e ingeniería.

## Decisión
Transformar el sitio de "catálogo de descargas" a "línea de tiempo evolutiva de herramientas con contexto personal".

## Características clave
- **Narrativa personal**: Cada herramienta incluye contexto de uso ("en mi tesis...", "para el curso...")
- **Descubrimiento**: Búsqueda por nombre, tags, categoría, año, década
- **Interactividad**: Comentarios por herramienta
- **Estático**: GitHub Pages, sin backend propio

## Consecuencias
- Se abandona el enfoque de "enlaces de descarga directa"
- Se prioriza el contenido histórico sobre la utilidad práctica actual
- El sitio se convierte en portfolio/documentación personal más que en utilidad para terceros
