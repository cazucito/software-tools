Lotus 1-2-3 fue mi primera hoja de cálculo seria. Antes de Excel dominar el mundo, Lotus era el estándar de facto para cálculos de negocio y análisis de datos.

## El "1-2-3"

El nombre venía de sus tres funciones principales:

1. **Hoja de cálculo** (spreadsheet)
2. **Gráficos** integrados
3. **Base de datos** básica

## La interfaz

Lotus 1-2-3 corría en DOS con una interfaz de menús en la parte superior. No había mouse (al principio), todo se hacía con teclas de función:

- `/` para abrir el menú
- `F1` para ayuda
- `F2` para editar celda
- `F9` para recalcular

Aprender los atajos de teclado era esencial para ser productivo.

## Macros WYSIWYG

Lotus tenía un sistema de macros increíblemente poderoso. Podías:

- Grabar secuencias de comandos
- Crear menús personalizados
- Automatizar reportes complejos
- Construir aplicaciones completas dentro de la hoja

Hice un sistema de facturación completo usando macros de Lotus. Los usuarios nunca sabían que estaban usando una hoja de cálculo.

## Contexto personal

Usé Lotus 1-2-3 desde 1992 hasta 1996, cuando Windows 95 y Excel 5.0 lo hicieron obsoleto. La transición fue dolorosa porque mis macros de Lotus no funcionaban en Excel.

Tuvimos que migrar todo a Visual Basic para Excel, lo que me llevó semanas de trabajo. Fue mi primera experiencia con "vendor lock-in" y migración forzada de tecnología.

Lotus 1-2-3 representa la era dorada de las aplicaciones DOS dominantes que fueron destronadas por Windows.
