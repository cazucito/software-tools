Mozilla Firefox fue mi navegador principal desde 2004 hasta 2018. Nació de las cenizas de Netscape y rescató la web del monopolio de Internet Explorer.

## La guerra de navegadores

En 2004, Internet Explorer tenía más del 90% del mercado. La web estaba estancada porque Microsoft no innovaba.

Firefox cambió eso:

- **Rendimiento**: Más rápido que IE
- **Seguridad**: Menos vulnerabilidades
- **Estándares**: Soporte de CSS y JavaScript moderno
- **Extensiones**: Adblock, Firebug, y miles más
- **Open source**: Transparencia y comunidad

## Características revolucionarias

### Tabs
Aunque Opera tenía tabs primero, Firefox popularizó el concepto. Dejar de tener 15 ventanas de IE y tener 15 tabs en una ventana fue liberador.

### Extensiones
El sistema de extensiones de Firefox era poderoso:

- **Adblock Plus**: Sin publicidad
- **Firebug**: Debugging de JavaScript y CSS
- **Greasemonkey**: Scripts personalizados
- **Password managers**: Integración con servicios

### Developer Tools
Firebug, la extensión de debugging, eventualmente se integró en Firefox y sentó las bases para las DevTools modernas.

## El declive

Chrome llegó en 2008 y cambió el juego:

- Más rápido (motor V8)
- Más simple (menos opciones)
- Google lo promocionaba en todas partes
- Firefox se volvió lento y pesado

## Contexto personal

Usé Firefox desde la versión 1.0 en 2004 hasta 2018. Fue mi navegador durante:

- La era de AJAX y Web 2.0
- El auge de jQuery
- La transición a HTML5
- El inicio de la era mobile

En 2018, cambié a Brave por privacidad y performance, aunque sigo teniendo Firefox instalado para testing.

Firefox representa el poder del open source para competir contra gigantes corporativos y mantener la web abierta.
